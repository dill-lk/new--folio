# Loris Bukvic Portfolio

This repository contains the static export of the portfolio website for Loris Bukvic, an award-winning website and multimedia designer specializing in premium website design, user experience, and interactive storytelling.

## About

This site showcases the work and skills of Loris Bukvic. It is a static website built with Next.js and exported for static hosting.

## Tech Stack

- **Framework**: Next.js (Static Export)
- **Styling**: Tailwind CSS
- **Animation**: GSAP (implied from data attributes)

## Running Locally

Since this repository contains the static build files, you can serve it using any static file server.

### Using `serve`

1.  Make sure you have Node.js installed.
2.  Run the following command in the project root:

    ```bash
    npx serve .
    ```

3.  Open http://localhost:3000 in your browser.

## Deployment

This site is designed to be hosted on platforms like Vercel, Netlify, or GitHub Pages.
